# Tiru Brand Assets — App-ready draft

These files were recreated as clean SVG/CSS/JSON assets based on the uploaded JPG brand sheet.

## Recommended app usage

- `tiru-primary-logo.svg` — main header/logo use on light backgrounds.
- `tiru-icon.svg` — icon-only use where a compact mark is needed.
- `tiru-app-icon.svg` — app-style rounded icon reference.
- `tiru-primary-logo-dark.svg` — dark background logo/banner use.
- `tiru-brand-tokens.css` — CSS custom properties for future UI integration.
- `tiru-brand-tokens.json` — design token reference.

## Notes

The uploaded JPG is useful as a brand reference, but SVG is the best format for the app logo because it stays sharp at all sizes.
These SVGs are clean approximations and can be refined further during the brand integration task.
